# **App Name**: EcoCollect

## Core Features:

- User Authentication: Enable users to login and signup via OTP (one-time password) and Google OAuth for authentication.
- Profile Management: Allow users to create and manage their profiles, including name, phone number, address, and reward points.
- Waste Submission Form: Provide a form for users to submit waste, including photo upload, waste category selection (Plastic, E-waste, Paper, Metal, Organic), weight entry, and desired pickup date & time.
- AI Waste Identification: Use generative AI as a tool that allows the user to provide photo of waste, and use this data to identify its kind (plastic, metal, etc.) for optimized categorization and recycling recommendations.
- Rewards System: Use a reward points system where users earn points for each kilogram of waste recycled.
- Environmental Impact Tracker: Showcase a dashboard displaying the total waste recycled and estimated CO₂ emissions saved through user activity.
- Offline Waste Submission: Offer offline mode support, enabling users to submit waste information even without an active internet connection.

## Style Guidelines:

- Primary color: Forest green (#388E3C) to represent nature and sustainability.
- Background color: Light green (#F1F8E9), providing a clean and fresh aesthetic.
- Accent color: A vibrant orange (#FF5722) will be used for calls to action and to highlight important information, creating a sense of urgency and importance.
- Body and headline font: 'Inter', a sans-serif font for clear readability and a modern feel.
- Use eco-friendly icons related to recycling and sustainability to enhance visual communication.
- Implement a clean and responsive layout, optimizing the experience across various devices, including mobile and desktop.
- Add subtle animations for feedback on user interactions, such as submitting waste or redeeming rewards.
